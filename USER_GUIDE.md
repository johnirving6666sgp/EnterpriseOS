# 企业OS 小团队试用手册

## 1. 这个系统是做什么的

企业OS 是一个给团队使用的 Agent 工作台。每位同事都有自己的个人助理 Agent，可以把客户问题、会议纪要、现场信息、报价想法、文件或图片发给它。

系统会尽量把这些信息整理成：

- 客户跟进
- 商机线索
- 待办任务
- 报价草案
- 内部经验和知识沉淀

目标不是增加大家的填表工作，而是让日常交流自动变成可跟进的业务动作。

## 2. 登录后先看哪里

登录后默认进入「业务工作台」。

每天优先看这几块：

- 今日重点线索：外部机会 Agent 找到的招标、新闻或商机。
- 我的客户：自己负责或相关的客户。
- 我的待办：需要推进的任务。
- 待处理报价：需要补充参数或继续确认的报价草案。
- 内部广播/协作邀请：Jamie、系统 Agent 或同事发来的协作信息。
- Agent 学习状态：系统最近从使用中沉淀了哪些东西。

如果不知道从哪里开始，可以直接点首页的：

- 客户拜访
- 会议纪要
- 报价准备

系统会自动进入你的个人 Agent 对话。

## 3. 个人助理 Agent 怎么用

点击「我的 Agent」进入自己的私密对话空间。

你可以直接输入或语音说：

- 今天拜访了哪个客户
- 客户提出了什么问题
- 会议讨论了什么
- 需要谁配合
- 是否需要报价
- 有哪些技术参数不确定
- 上传会议纪要、PDF、图片或现场资料

建议不要一开始就整理得很完美，先把原始信息发给 Agent。Agent 会帮助你提取客户、需求、风险、任务和下一步动作。

## 4. 语音和文件怎么用

在对话框下方：

- 按住「按住说话」开始语音输入，松开结束。
- 点击「上传文件/图片」可以上传资料。
- 如果上传的是 PDF、文本、Markdown、CSV，系统会尽量读取内容。
- 图片和较大的文件会先作为附件记录，后续可以继续补充说明。

## 5. Agent 回复后要做什么

每条 Agent 回复下面有几个按钮：

- 生成任务：把这条回复转成任务卡。
- 有用：告诉系统这类回答值得保留。
- 不准：告诉系统这个回答方向不对。
- 需更具体：告诉系统下次要给出更明确的步骤、参数或依据。

请大家尽量点反馈。反馈越多，Agent 越容易变成真正懂业务的助手。

## 6. 商机线索怎么流转

外部机会 Agent 会扫描招标网站、行业新闻和相关企业信息。

线索进入「商机雷达」后，会有 AI 评分：

- 需求真实性
- 预算可能性
- 采购紧急度
- 公司匹配度
- 推荐负责人
- 下一步动作

看到有价值的线索，可以点「收藏至我的助理」。

收藏后，这条线索会进入你的个人 Agent 对话空间，你可以继续问：

- 这个商机是否值得跟？
- 应该先联系谁？
- 需要准备哪些技术资料？
- 是否需要报价？
- 下一步任务怎么分配？

## 7. 客户管理怎么用

「客户管理」按阶段展示客户：

未接触 → 已接触 → 有意向 → 待报价 → 待成交 → 已成交

客户管理 Agent 会根据对话、任务、报价和商机，帮助整理：

- 客户名称
- 当前阶段
- 负责人
- 最近沟通
- 下一步动作
- 关联任务
- 关联报价
- 关联商机

大家不用重复维护太多字段，先把真实沟通发给个人 Agent，系统会逐步整理。

## 8. 任务看板怎么用

任务来源包括：

- 个人 Agent 对话
- 会议纪要
- 客户访谈
- 商机收藏
- 广播反馈
- 报价流程

每个任务尽量包含：

- 任务标题
- 负责人
- 截止时间
- 来源
- 关联客户/商机/报价
- 当前状态
- 结果评估

任务状态包括：

- 待办
- 进行中
- 等待反馈
- 已完成
- 已关闭

任务完成后，系统会把结果交给内部信息 Agent 做复盘沉淀。

## 9. 报价 Agent 怎么用

报价 Agent 不是直接给最终价格，而是帮助形成可信的内部报价方案。

它会尽量整理：

- 报价对象
- 设备或服务范围
- 技术参数
- 成本构成
- 风险点
- 历史报价参考
- 市场价格参考
- 推荐报价区间
- 可谈判空间
- 需要人工确认的问题

正式对外报价前，仍需要负责人确认。

## 10. 内部广播和协作邀请

「广播中心」用于发布工作计划、商机、报价提醒或讨论邀请。

收到广播后，请及时点：

- 收到
- 跟进中
- 需要讨论

如果点「需要讨论」，需要指定一个或多个同事。系统会生成讨论邀请和后续任务。

广播会显示几人已读、谁已读、谁反馈了什么。

## 11. 隐私和权限

每位同事只能进入自己的个人 Agent 空间。

普通同事看不到其他人的私密对话。

系统会把有价值的信息抽象成经验和知识，但不会把其他同事的原始私密聊天直接展示给大家。

如果同事离职，管理员可以中止账号权限，并把该同事的 Agent 资产移交给接替人员。

## 12. 每天建议怎么用

建议每位同事每天做三件事：

1. 打开业务工作台，看今日线索、待办和广播。
2. 把当天客户沟通、会议纪要或问题发给自己的 Agent。
3. 对 Agent 回复点一次反馈：有用、不准或需更具体。

只要大家持续输入真实业务信息，系统就会逐步沉淀客户经验、报价经验、设备知识、材料知识和商机判断规则。
